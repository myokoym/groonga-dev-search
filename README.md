# groonga-dev-search

A full-text search tool for groonga-dev mailing list.

## Usage

### crawl

    % ruby crawler.rb

### index

    % ruby indexer.rb

### search (temporary)

    % gem install groonga-database-viewer-gtk
    % groonga-database-viewer-gtk db/groonga-dev-search.db
